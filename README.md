# Projet Front-End : Interface de Portail Captif

Ce projet est une application web moderne construite avec Next.js et TypeScript, conçue pour fournir une interface utilisateur riche et interactive pour la création et la gestion de formulaires pour  portail captif déjà en place. 

## Fonctionnalités Principales

*   **Page d'atterrissage :** Une page d'accueil pour présenter l'application et ses fonctionnalités, avec des appels à l'action pour l'inscription.
*   **Authentification des utilisateurs :** Un flux d'authentification complet et sécurisé comprenant l'inscription, la connexion, la réinitialisation du mot de passe avec vérification par OTP (One-Time Password).
*   **Processus d'intégration (Onboarding) :** Un guide en deux étapes pour aider les nouveaux utilisateurs à configurer leur profil et à comprendre l'application.
*   **Tableau de bord (Dashboard) :** Une interface utilisateur épurée et intuitive qui affiche les métriques clés, les formulaires créés et les analyses de données.
*   **Constructeur de Formulaires (Form Builder) :** Un outil puissant permettant de créer des formulaires simplement en décrivant les champs. Il intègre des suggestions de questions pour optimiser les taux de conversion.
*   **Analyses en temps réel :** Des graphiques et des visualisations pour suivre les prospects collectés, les tendances et les sources de trafic en temps réel.

## Technologies et Outils

*   **Framework :** [Next.js](https://nextjs.org/)
*   **Langage :** [TypeScript](https://www.typescriptlang.org/)
*   **Style :** [Tailwind CSS](https://tailwindcss.com/) pour le design d'interface.
*   **Composants d'UI :** [Shadcn/ui](https://ui.shadcn.com/) - Une collection de composants réutilisables.
*   **Gestion de formulaires :** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/) pour la validation.
*   **Icônes :** [Lucide React](https://lucide.dev/guide/packages/lucide-react)

## Guide de Style

Le design de l'application suit les directives suivantes pour une expérience utilisateur cohérente et professionnelle :

*   **Couleur primaire :** Bleu profond (`#1E3A8A`)
*   **Couleur de fond :** Gris clair (`#F9FAFA`)
*   **Couleur d'accent :** Sarcelle (`#34D399`)
*   **Police des titres :** 'Space Grotesk'
*   **Police du corps de texte :** 'Inter'

## Démarrage Rapide

Pour lancer le projet en local, suivez ces étapes :

1.  **Installation des dépendances :**
    ```bash
    npm install
    ```

2.  **Lancement du serveur de développement :**
    ```bash
    npm run dev
    ```

L'application sera alors accessible à l'adresse `http://localhost:9002`.
